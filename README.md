# Our Team - Responsive Web Page

This project is a simple **responsive team page** built for a Web Programming assignment.  
It showcases a fictional company's team members using **HTML5**, **CSS3**, and **Bootstrap 5**.

- A **Team section** with:
  - Three team members displayed in responsive Bootstrap cards
  - Custom images for each member
  - Name, role (CEO, CTO, Project Manager)
  - Short description
  - Social media icons (LinkedIn, GitHub, etc.)
- A **Contact section** with:
  - Simple contact text
  - Email link

Custom CSS is used to give the page a **modern, dark-themed, glassmorphism-style design**.
